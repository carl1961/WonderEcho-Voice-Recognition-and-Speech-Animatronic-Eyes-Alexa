# CI130X SDK Introduction

## Overview

CI130X SDK Suitable for CI130X series chips

## SDK  Directory Structure

| Contents   | Description                    |
| ---------- | ----------------               |
| .vscode    | Compiler configuration file    |
| components | component                      |
| driver     | Driver                         |
| libs       | Library files                  |
| projects   | Sample projects                |
| startup    | Startup files                  |
| system     | System files                   |
| tool       | Firmware build & upgrade tools |
| utils      | Debug helper tools             |
